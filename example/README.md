# Example Moonbar Config

> This is an example config. screenshots below

## Installation

Install the files found in this directorry to:
`$XDG_CONFIG_HOME/bar/` or `~/.config/bar/`

### init.lua

The [init.lua](init.lua) file contains all the necessary lua code to ceate the bar shown below.

### style.css

The [style.css](style.css) file contains the styles for the bar. They were generated w/ lessc so....good luck
